# Product Overview

This is a Java Maven application demonstrating modern CI/CD practices with GitHub Actions. The application has evolved from a basic "Hello World" program to include a comprehensive user authentication system with web and CLI interfaces.

## Core Features

- **Basic Application**: Simple Java application with "Hello World" functionality
- **User Authentication System**: Complete authentication module with user management, session handling, and password reset capabilities
- **CI/CD Integration**: GitHub Actions pipeline for automated builds, testing, and quality checks

## Purpose

This application demonstrates:
- Maven-based Java project structure
- Authentication system architecture with security best practices
- Test-driven development with comprehensive unit and integration tests
- Modern CI/CD pipeline implementation with GitHub Actions
- Web-based and CLI interfaces for the same backend system
